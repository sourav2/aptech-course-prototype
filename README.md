# Aptech Aviation: Dynamic Course Finder
A mobile-first prototype featuring a "Not Sure?" helper that uses dynamic filtering logic to match students with the right aviation and hospitality courses.

## 🎯 The UX Challenge
Choosing a career in aviation can be overwhelming. I designed this to solve "Choice Paralysis" by replacing a static list with an interactive discovery flow. 

## 🧠 Technical Logic (The "Why")
I used **Vanilla JavaScript** to build a lightweight recommendation engine:

* **Dynamic Filtering:** Instead of hard-coded pages, the "Not Sure?" helper filters an **Array of Course Objects** based on user interest (e.g., 'Ground Staff' vs 'Cabin Crew').
* **Conditional Rendering:** The UI updates instantly to show relevant modules without a page reload, maintaining user momentum.
* **Search Optimization:** Implemented a simple keyword-matching logic to help users find specific certifications quickly.
* **Mobile-First Interaction:** Designed with large tap targets and bottom-sheet inspired navigation for easy one-handed use.

## 🛠️ Tech Stack
* **HTML5/CSS3:** Custom responsive grid and glassmorphism UI elements.
* **Vanilla JavaScript:** Array manipulation, event delegation, and DOM updates.
